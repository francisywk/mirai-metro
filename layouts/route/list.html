About Station Code

Station code is consist of a letter representing the metro line and a number for the station. They are circled with a color that represents the metro line making it easy to find your station.

Overview

R2 B5
  • The alphabet letter represents the metro line name.
  • The number represents the station number of the metro line.
  • A single station may have multiple designated station codes of different lines.
  • Station code are circled with a color that represents the metro line.

Why using?

  • It is easy to know the station code for the metro line of your desired station.
  • It is easy to know the number of stations until your desired or transfer station.
  • It is easy to remember or search for your desired station instead of remembering station names.

Where to find?

Station codes are displayed alongside station names on station banner.
